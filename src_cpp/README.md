# Trilha C++

Escolha esta pasta apenas se voce decidir implementar a atividade em `C++`.

Crie aqui, no minimo:

- `main.cpp`
- `equipamento.hpp`
- `equipamento.cpp`
- `sensor_temperatura.hpp`
- `sensor_temperatura.cpp`

Use o diagrama final como contrato da sua implementacao.

Comando esperado ao final:

```bash
g++ -std=c++17 -Wall -Wextra -O2 src_cpp/main.cpp src_cpp/equipamento.cpp src_cpp/sensor_temperatura.cpp -o laboratorio
./laboratorio
```
