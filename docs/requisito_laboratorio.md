# Requisito do Laboratorio

O laboratorio de ensaios precisa registrar seus equipamentos e acompanhar sensores de temperatura.

Cada equipamento possui identificacao, descricao e estado operacional.

Alguns equipamentos possuem sensores acoplados, e cada sensor registra pelo menos uma leitura atual.

Quando um equipamento precisa de verificacao metrologica, deve existir uma ordem de calibracao com data, responsavel e status.

O sistema tambem precisa representar o tecnico que executa calibracoes e associar esse tecnico as ordens quando necessario.

Use este requisito como ponto de partida.
Voce pode ajustar nomes, classes e relacoes, desde que justifique suas decisoes no diagrama final.
